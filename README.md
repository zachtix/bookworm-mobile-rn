# Bookworm Mobile App (React Native)

Mobile application recommendation book

## Features

-   Recommendation Book
-   Add New Book
-   View Your Book
-   Delete Your Book

## Tech Stack

-   React Native
-   Axios
-   Zustand

## Installation

``` bash
git clone https://github.com/zachtix/bookworm-mobile-rn.git
cd bookworm-mobile-rn
npm install
npx expo start
```

Make sure backend API [(bookworm-api-rn)]([https://expo.dev](https://github.com/zachtix/bookworm-api-rn)) is running.
